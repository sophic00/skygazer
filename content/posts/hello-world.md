Welcome to my new blog! This is a quick test post to ensure the static site generator is compiling and displaying pages correctly. 

This blog is built using **Zola**, one of the fastest and most lightweight static site generators on the planet, paired with custom responsive minimalist styling.

### What is the goal here?

The goal is to keep things simple, fast, and completely focused on content:
- **No bloat**: No heavy JavaScript libraries or heavy web trackers.
- **Fast rendering**: Instant loading times.
- **Clean design**: Focusing on readability above all else.

Stay tuned for more deep dives, technical tutorials, and thoughts on minimalist software engineering!
